StarterCar

StarterCar utilizes ajax and php to asynchronously gather data from a mysql database in order to provide auto-completing search results and data without ever having to refresh the page. Bootstrap handles responsive design to ensure it looks good on any platform.

OpenWeatherMap

OpenWeatherMap is a weather data/forecasting API that provides day-to-day weather data for locations all over the world. JavaScript is used to get and parse the JSON data, which is then displayed in a 5 day forecast format. Top weather news from around the world is also grabbed in a similar fashion from reddit.com/r/weather and displayed below the forecast data.

Prioritize

Prioritize is an extension of Simple To-Do. The main functionality is the same, however prioritize utilizes php nd MySQL on the backend to allow users to sign up and save multiple lists to a database. This way, databases are safe from clearing browser data, and accessible from any device a user logs in from.
